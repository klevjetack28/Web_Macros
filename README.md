# Overview

Web Macros is the second version of the Roku Macros project. After I completed the engineering goals I wanted with reverse engineering, making something with basic electronics, and programming an Arduino, I did more research into more efficient ways to communicate with the TV and found Roku has an API I can communicate with to accomplish the same goal. The best part about the API is I have more freedom and options on what I am able to accomplish. A single example is typing is possible through simple API calls instead of complex keyboard mappings, and algorithms to find the input to the next key, etc.
